## Dependencies

- [Quantum Voxel](https://gitlab.com/ultreon/quantum-voxel) and [QuantumJS](https://gitlab.com/ultreon/quantum-js/quantumjs)

## How to build

```bash
npm install
npm run build
```

The output is located in the `build` directory.
Zip it up to use it in Quantum Voxel.

## How to run

Put the zipped file in the mods directory and restart Quantum Voxel if it's running.
